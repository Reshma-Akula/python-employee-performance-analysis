# 🐍 Employee Performance & Project Cost Analysis using Python

## 📌 Project Overview

This project analyzes employee performance and project information using Python, Pandas, and NumPy. The objective is to clean, transform, and analyze employee and project datasets to generate meaningful business insights.

---

## 🎯 Business Objective

The project demonstrates how Python can be used for:

- Data Cleaning
- Data Transformation
- Missing Value Treatment
- Employee Performance Analysis
- Project Cost Analysis
- Bonus Calculation
- Employee Promotion & Demotion Analysis
- Data Aggregation
- Business Reporting

---

## 🛠 Tools & Libraries

- Python
- Pandas
- NumPy
- Jupyter Notebook

---

## 📂 Dataset

The project uses three datasets:

- Employee Data
- Project Data
- Seniority Data

---

## 🔍 Tasks Performed

✔ Created DataFrames from CSV files

✔ Handled missing values using running averages

✔ Split employee names into First Name and Last Name

✔ Merged multiple datasets

✔ Calculated employee bonuses

✔ Updated employee designation levels

✔ Added title prefixes (Mr./Mrs.)

✔ Promoted employees based on age criteria

✔ Calculated total project cost per employee

✔ Filtered employee records based on city
